# AI Avatar Domain Finder

An interactive domain search tool for finding the perfect domain name for AI video avatar apps.

## Features

- **160+ Domain Suggestions** across multiple categories
- **Smart Filtering** by category, TLD, status, and favorites
- **Domain Availability Checking** using DNS lookup
- **Favorites System** with localStorage persistence
- **Multiple Categories**: Premium, AI-focused, Video/Chat, Influencer, Creative, Brandable, Playful, Tech-forward, Social
- **Various TLDs**: .ai, .com, .io, .app, .tv, .chat, .live, .video, .social

## How to Use

1. Browse through 160+ curated domain suggestions
2. Filter by category, TLD, or search for specific terms
3. Click "Check" to verify domain availability
4. Click the heart icon to favorite domains you like
5. Use "Show Favorites Only" to view your favorited domains
6. Use "Check All Domains" to batch check availability

## Categories

- **Premium Names**: Short, memorable, high-value domains
- **AI Focused**: Emphasizes AI and virtual technology
- **Video/Chat**: Video calling and chat focused
- **Influencer Theme**: Celebrity and creator focused
- **Creative/Unique**: Imaginative and distinctive names
- **Brandable**: Strong brand potential
- **Playful/Fun**: Casual and energetic
- **Tech Forward**: Cutting-edge technology focus
- **Social/Connect**: Social and connection focused
- **Short & Memorable**: Ultra-short, easy to remember

## Technology

Built with vanilla HTML, CSS, and JavaScript. Uses:
- Google DNS API for domain availability checking
- localStorage for persisting favorites
- Responsive CSS Grid layout

## Use Case

Designed for AI video avatar apps where users can have video chats with virtual versions of influencers and personalities, simulating a FaceTime-like experience.
